# Images Directory

This directory contains all image assets for the WMB Fırın website.

## Image Assets

All images should be optimized and placed in this directory. Currently, images are loaded directly from Unsplash URLs for performance and convenience.

### Recommended Image Specifications

- **Format**: WebP (preferred) or JPG
- **Quality**: 80-90% for optimal balance
- **Dimensions**: 
  - Hero images: 2070x1380px (3:2 ratio)
  - Menu cards: 800x1000px (4:5 ratio)
  - Blog thumbnails: 1200x675px (16:9 ratio)
  - Signature sections: 2070x1200px

### Current Image Sources (Unsplash)

To download and optimize images locally, use the following URLs:

- Hero: https://images.unsplash.com/photo-1414235077428-338989a2e8c0
- Menu items: Various bakery/food images from Unsplash
- Signature: Bakery display images
- Heritage: Baker kneading dough images
- Franchise: Professional bakery kitchen images

### Color Grading

All images should have a warm, slightly desaturated bakery tone applied via CSS (`img-bakery` class):
- Brightness: 95%
- Contrast: 105%
- Saturation: 90%

### Alt Text Guidelines

All images must have descriptive alt text in the same language as the surrounding content:
- Turkish: Descriptive text in Turkish
- English: Descriptive text in English
